# ENSA_Inversion_Controle

Bienvenue dans le dépôt **ENSA_Inversion_Controle** !  

Ce dépôt contient des projets et des applications liés au domaine de **l'ingénierie logicielle**, avec un focus particulier sur le **fonctionnement détaillé du code en Java**. Il a pour objectif de centraliser des travaux pratiques, des exemples, et des applications concrètes pour mieux comprendre les concepts fondamentaux de l'ingénierie logicielle.

## Contenu du dépôt

- Projets Java illustrant des concepts clés de l'ingénierie logicielle.
- Codes sources détaillés pour comprendre le fonctionnement interne des applications.
- Exercices et exemples pratiques pour l'apprentissage et la maîtrise des bonnes pratiques de programmation.
- Documentation sur le fonctionnement du code et les structures logicielles utilisées.

## Objectifs

1. Fournir une base de référence pour l’apprentissage de l’ingénierie logicielle en Java.
2. Illustrer les concepts théoriques par des applications concrètes.
3. Favoriser la compréhension du code, son architecture et son fonctionnement interne.
4. Permettre aux étudiants et développeurs de réutiliser et d’adapter le code pour leurs propres projets.

## Technologies utilisées

- **Java** : principal langage de programmation pour les projets.
- **Maven/Gradle** (selon les projets) pour la gestion des dépendances et la compilation.
- Documentation et commentaires détaillés pour chaque projet.
